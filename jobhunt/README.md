# jobhunt

Automated job hunting dashboard for game industry roles — scrapes Greenhouse, Lever, EA, Indeed, Glassdoor · detects Gmail replies · generates cover letters with Claude AI.

## Setup (5 steps)

**1. Clone & enter the folder**
```bash
cd jobhunt
```

**2. Set your Anthropic API key** (for cover letter generation)
```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

**3. (Optional) Enable Gmail integration**
- Go to [console.cloud.google.com](https://console.cloud.google.com) → new project → enable **Gmail API**
- OAuth consent screen → External → add your Gmail as test user
- Credentials → OAuth 2.0 Client ID → Desktop App → download JSON
- Rename it `credentials.json` and place it in this folder

**4. Run**
```bash
bash run.sh
```
This installs deps, runs Gmail OAuth (if credentials.json present), starts the scraper, and opens the dashboard.

**5. Open** [http://localhost:8000](http://localhost:8000)

## What it does

| Feature | Details |
|---|---|
| Scrapes | Greenhouse API, Lever API, EA Workday, Blizzard, Indeed, Glassdoor, 2K |
| Filters | Cinematic Artist, Video Editor, Motion Designer, Social Media Editor (Jr/Mid/Sr) |
| Gmail | Detects replies from companies you applied to every 30min |
| Cover letters | Generated by Claude per job, in English or Portuguese based on region |
| Schedule | Scraper every 6h · Gmail check every 30min (auto, via APScheduler) |

## Target studios

Riot Games · EA · Blizzard · 2K · Greenhouse boards · Lever boards

Add more in `config.py` → `GREENHOUSE_BOARDS` or `WORKDAY_TARGETS`.
